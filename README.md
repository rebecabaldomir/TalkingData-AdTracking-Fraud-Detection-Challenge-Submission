# TalkingData AdTracking Fraud Detection Challenge
#### Rebeca Andrade Baldomir
Abril 2018

Competição da plataforma Kaggle (https://www.kaggle.com/c/talkingdata-adtracking-fraud-detection) que visa predizer se um usuário irá fazer o download de um aplicativo após clicar em um ad online utilizando algoritmo de classificação LightGBM.
